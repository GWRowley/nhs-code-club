For the most part, this section is pretty straight forward. The only thing to consider, which is likely only for a developer to worry about is whether to use the HTML or Nunjucks when using a component from the component library.

The prototype kit which you will be using allows you to use both, however some other projects that might be built in Java or static HTML sites you may be limited to using just the HTML blocks of code.

For the sake of simplicity of this tutorial going forward we will use HTML.

To use the components head to the 'component library' which can be accessed from the service manual (opens in a new tab), select the relevant component from the menu, and copy and paste the block of code into the relevent temlpate.

For example, if I wanted to add a 'Date input' to this page, I would copy the HTML for a date input from the service manual, find the template for this page found in app/views/nhs-components.html and paste it.

Please feel free to open the file mentioned above in your code editor and paste in a component of your choice.
